# VAT API (vatapi)

EU and UK VAT compliance REST API providing VAT rate retrieval, VAT number validation via the European Commission VIES system and UK HMRC, currency conversion, and VAT-compliant invoice generation. Operated by Eventured Ltd and launched in 2015, it connects directly to authoritative EU and HMRC sources to deliver audit-ready consultation numbers and real-time rate data.

## APIs

### VAT API v2

REST API for EU and UK VAT compliance including VAT rate lookups by country, VAT number validation against VIES and HMRC, currency conversion rates from HMRC and ECB, and generation of VAT-compliant invoices with sequential numbering.

- **Human URL:** [https://vatapi.com/](https://vatapi.com/)
- **Base URL:** `https://eu.vatapi.com/v2`
